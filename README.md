# Bootstrap Toggle Sidebar
Bootstrap doesn't provide a sidebar component. This example is using bootstrap ".navbar" and ".nav", "data-toggle" to implement a sidebar component

# How it works
```
    $ sass ./scss/style.scss ./css/style.css 
```
